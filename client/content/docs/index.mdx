# Welcome to Run Flare

Run Flare is a powerful, secure, and high-performance code execution engine designed for developers, educators, and platform builders. It allows you to execute code in multiple languages, handle batch submissions, and receive real-time updates via WebSockets.

## Key Features

<Cards>
  <Card title="Multi-language Support" href="/docs/getLanguages">
    Execute code in a wide range of popular programming languages including Python, JavaScript, C++, Java, and more.
  </Card>
  <Card title="Real-time Updates" href="/docs/connectWebSocket">
    Receive instant feedback on code execution status and results through our robust WebSocket integration.
  </Card>
  <Card title="Batch Processing" href="/docs/createBatchSubmissions">
    Submit and process multiple code snippets simultaneously for efficient bulk execution.
  </Card>
  <Card title="Secure Sandboxing" href="#">
    Every execution runs in a strictly isolated environment, ensuring security and preventing malicious activity.
  </Card>
</Cards>
